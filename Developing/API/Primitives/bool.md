# bool

The boolean class for Cheddar

#### Arguments
| name | type | description |
| ---- | ---- | ----------- |
| `value` | any | the value, can be a cheddar class. (will be cast to boolean if applicable)

#### Usage:
`true`, with api:
```js
cheddar.init(
    cheddar.bool,
    true
)
```
---
`false`, with api:
```js
cheddar.init(
    cheddar.bool,
    false
)
```
